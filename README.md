# spark-hdfs-demo
A project for execution of Word Count problem on Spark by referring data stored on hdfs.

# Steps
1. Install Git
```
sudo apt update
sudo apt install git
```
2. Clone this repository
```
git clone https://github.com/anshudmp/spark-hdfs-demo.git
```
3. Run install-deps
```
sh install-deps.sh
```
